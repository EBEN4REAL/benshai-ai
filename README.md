## Development
1. Install dependencies
```sh
npm install
```
2. Run the project in development mode
```
npm run dev
```


## Run tests
```sh
Run unit texts once
npm run test:unit
```
You can watch for changes with
```sh
npm run test:unit -- --watch
```

## Build production
Build for production
```
npm run build
```
